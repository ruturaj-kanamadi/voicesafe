About VoiceSafe


VoiceSafe is a web-based smart complaint management system for water-related issues integrated with IoT-enabled automatic leakage detection.


The system allows citizens to register complaints related to water leakage, water quality, and supply disruptions through a website.


In addition, a flow sensor connected to an ESP32 microcontroller automatically detects leakage using a threshold-based model (flow below 80% of calibrated normal flow) and registers complaints through the web platform.


Existing complaint systems rely mainly on manual reporting and delayed response. VoiceSafe combines manual complaint reporting with proactive sensor-triggered alerts.


Technologies Used


  • Frontend: HTML, CSS
  • Backend: Python Flask
  • Database: MySQL
  • Hardware: Flow Sensor + ESP32

Results show successful complaint registration from users and simulated automatic complaint generation through sensor logic.


The system demonstrates low-cost smart water monitoring with future scope for smart city applications.
